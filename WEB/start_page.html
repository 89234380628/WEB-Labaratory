Урок 1. Введение в WEB - разработку

Для того,чтобы начать говорить про WEB - разработку и про WEB - технологии в целом необходимо понимать, а так же знать следующие понятия:

Интернет – это всемирная компьютерная сеть, предназначенная для хранения, обработки и передачи информации.

Домен - это буквенное отображение вашего сайта, по которому пользователь вас находит в сети. Вы покупаете его на определенный срок у Регистратора (компания, которая заведует доменами).

Пример

Один из самых известных ресторанов быстрого питания в России имеет следующий домен: kfc.ru

Хостинг - то место на диске некоего сервера, где лежит ваш сайт. Вы арендуете его на время у хостера (компания, которая предлагает услуги хостинга) для того, чтобы ваш сайт был виден в сети.

SSL (HTTP/S) сертификат - цифровой сертификат, удостоверяющий подлинность веб-сайта и позволяющий использовать зашифрованное соединение. Аббревиатура SSL означает Secure Sockets Layer – протокол безопасности, создающий зашифрованное соединение между веб-сервером и веб-браузером.

Процесс работы SSL сертификата:

  1. Браузер или сервер пытается подключиться к веб-сайту (веб-серверу), защищенному с помощью SSL
  2. Браузер или сервер запрашивает идентификацию у веб-сервера
  3. В ответ веб-сервер отправляет браузеру или серверу копию своего SSL-сертификата
  4. Браузер или сервер проверяет, является ли этот SSL-сертификат доверенным. Если это так, он сообщает об этом веб-серверу
  5. Затем веб-сервер возвращает подтверждение с цифровой подписью и начинает сеанс, зашифрованный с использованием SSL
  6. Зашифрованные данные используются совместно браузером или сервером и веб-сервером

URL - Это адрес страницы в сети. URL состоит из протокола, домена и пути к странице.

Пример

URL: https://www.kfc.ru/restaurants, где:

https:// - протокол

www.kfc.ru - Домен

/restaurants - путь к странице

Система управления контентом сайта (CMS) - программное обеспечение для быстрой публикации текста, картинок и других материалов на сайте.

Важно

Наиболее популярная CMS - Wordpress. Она позволяет без программирования создавать довольно функциональные сайты.

При этом у Wordpress есть и свои минусы: Низкое быстродействие, "дыры" (ошибки) безопасности в плагинах.

Движок сайта - программное обеспечение, на котором работает сайт.

Важно

Движком может быть типовая система управления контентом (CMS), например: Wordpress.

Либо это может быть самописная система (созданная специально под проект на некотором фреймворке).

Язык программирования - способ описания работы приложения в виде специального текста.

Технологии веб-разработки

Front-end (интерфейсные) технологии:

Интерфейсные технологии предназначены для «клиентской стороны» вашего веб-сайта или приложения. Они используются для разработки интерактивных компонентов вашего сайта и создания элементов, которые пользователи видят и с которыми взаимодействуют. Сюда входят цвета и стили текста, изображения, кнопки и меню навигации.

HTML

HTML - язык гипертекстовой разметки.


Это одна из фундаментальных технологий, необходимых для веб-разработки. Он обеспечивает базовую структуру веб-страницы.

Код HTML гарантирует, что весь контент на сайте правильно отформатирован. Это сделано для того, чтобы ваш интернет-браузер мог отображать контент, как задумано. Без HTML браузер не может отображать текст или загружать изображения и другие элементы.

CSS

Каскадные таблицы стилей (CSS) - формальный язык описания внешнего вида документа, написанного с использованием языка разметки HTML.


В то время как HTML используется для структурирования веб-страницы, CSS определяет внешний вид этой структуры. Это включает макеты страниц, цвета, шрифты и расположение элементов.

Back-end (серверные) технологии:

Внутренние технологии предназначены для «серверной части» вашего сайта или приложения. Они хранят и упорядочивают данные и следят за тем, чтобы на интерфейсе всё работало.

Например, когда пользователь предоставляет учетные данные для входа в приложение социальной сети, используются внутренние технологии для проверки правильности этих учетных данных. После проверки учетных данных сервер отправит обратно имя профиля, изображение и другую связанную информацию.

JavaScript (JS)

JavaScript - это язык управления элементами на сайте. Он умеет красить кнопки, запускать анимации, загружать эффекты, и всё это — без перезагрузки страницы.


Для реализации определенных задач, этот язык программирования можно использовать как и для интерфейсной, так и для серверной части вашего WEB - сайта.

PHP

PHP - это серверный язык программирования, предназначенный "оживлять" HTML страницы.


РНР позволяет изменять веб-страницу на сервере непосредственно перед тем, как она будет отправлена браузеру в отличии от HTML который показывает статичные данные и не может их менять.


В ходе исполнения PHP может изменить или динамически создать любой HTML-код, который и является результатом исполнения сценария. Затем сервер отправляет этот код браузеру. При этом браузеру не известно, как была сформирована данная страница — статично сверстана верстальщиком, или динамически создана при участии PHP. Это не важно, т.к. браузер всегда работает только с тем, что получил от сервера.

Python

Python - универсальный язык программирования, применимый в том числе и в WEB. С технической точки зрения web-приложение на Python — полноценное приложение, загруженное в память, обладающее своим внутренним состоянием, сохраняемым от запроса к запросу.


При использовании PHP возникают ряд проблем приобработке данных, так как он "живет" только при выполнении запросов. В отличие от него, Python же web - приложение, которое постоянно находится в память, обрабатывая множество запросов без "перезагрузки" страницы, тем самым позволяет исключить проблеммы при обработки данных в PHP.

Rust

Rust - это новый язык, ориентированный на безопасность и особенно безопасный параллелизм.


Параллелизм в разработке программного обеспечения означает набор методов, которые позволяют компьютерной программе выполнять несколько различных задач одновременно. Безопасный параллелизм дает вам эффективность, необходимую для выполнения нескольких задач, сохраняя при этом безопасность вашего кода.
